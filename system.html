Введение

Назначение системы

Приложение «Игра «Морской бой» предназначено для игры пользователя с компьютерным противником, использующим определённые стратегии обстрела в зависимости от выбранного уровня сложности.предназначено для игры с компьютерным противником, использующим разные стратегии обстрела в зависимости от выбранного уровня сложности.

В системе реализована одна роль – игрока.

Перед началом игры пользователю необходимо авторизоваться. Пользователь может расставить корабли вручную, автоматически по заданному алгоритму или загрузить поле из файла, а также сохранить свою расстановку. Пользователю доступны три уровня сложности, которые определяют стратегию обстрела компьютерного противника.

В настольном приложении также есть возможность получения справочной информации как о самом приложении, так и о разработчиках.

Условия работы системы

Для корректного функционирования приложения необходимы следующие минимальные характеристики:

  • Тип ЭВМ: x86-64 совместимый;
  • Объем ОЗУ – не менее 5 Гб;
  • Объем свободного дискового пространства – не менее 20 Гб;
  • Устройства ввода: клавиатура и мышь;
  • Процессор – Intel(R) Core(TM) i5-10300H или эквивалентный, частота не менее 2.50 ГГц;
  • Дисплей с разрешением не менее 1024 × 768 пикселей;
  • Операционная система Windows 10 и выше;
  • Python версии 3.11 и выше;
  • Библиотека: Pygame 2.3.1.

Установка системы

Для запуска настольного приложения «Морской бой» необходимо либо склонировать репозиторий с GitHub, либо получить необходимые файлы в виде zip-архива.

Убедиться, что на компьютере установлен Python, и переменная окружения PATH настроена.

Затем используя любой редактор или IDE, открыть терминал PowerShell и выполнить следующие команды:

Шаг 1: Создание виртуального окружения

python -m venv venv

Эта команда создаст новую папку venv в проекте, где будет находиться виртуальное окружение.

Шаг 2: Активация виртуального окружения

.\venv\Scripts\Activate

После активации виртуального окружения префикс в терминале изменится, и можно буудет увидеть (venv), который указывает на активированное окружение.

Шаг 3: Установка библиотеки Pygame

После активации виртуального окружения необходимо установить библиотеку Pygame (так как Pygame не входит в стандартную библиотеку Python) 

pip install pygame

Работа с системой

Создание профиля

Для того чтобы начать игру пользователю необходимо создать профиль, пока профиль не создан, кнопка «Начать игру» будет недоступна. (см. рисунок ниже).

Для создания профиля необходимо выполнить следующие шаги:

1. Выбрать изображение для аватара из предложенных десяти предустановленных вариантов.

Если пользователь не выбрал аватар и нажал кнопку «Выбрать», то внизу окна будет располагаться соответствующее сообщение (см. рисунок ниже).

2. Придумать логин, который должен содержать от 4 до 10 символов.

Если пользователь при вводе логина не соблюдает допустимую длину, то под полем ввода логина будет располагаться соответствующее сообщение (см. рисунок ниже).

Оба шага являются обязательными для завершения создания профиля. Если хотя бы один из шагов не выполнен, система не позволит начать игру и внизу окна создания профиля будет располагаться соответствующее сообщение (см. рисунок ниже).

После нажатия на кнопку «Сохранить профиль» осуществляется проверка введённых данных, при успешном завершении которой, пользователь перенаправляется на начальное окно приложения и может нажать на кнопку «Начать игру»

Настройка параметров игры

Выбор уровня сложности компьютерного противника

После создания профиля пользователю необходимо выбрать уровень сложности для игры с компьютерным противником. Для этого нужно нажать соответствующую кнопку, которая отображает доступные варианты сложности. (см. рисунок ниже).

Выбор способа расстановки кораблей

Ручная расстановка

Для того чтобы расставить корабли вручную пользователю необходимо выбрать соотвествующий шаблон корабля под своим полем и перенести удерживая левой кнопкой мыши на поле. (см. рисунок ниже).

Всего доступно 4 шаблона кораблей:

  • четырёхпалубный корабль – в количестве 1 ;
  • трёхпалубных корабль – в количестве 2;
  • двухпалубных корабль– в количестве 3;
  • однопалубных корабль – в количестве 4;

На поле предусмотрено изменение ориентации корабля при ручной расстановке кораблей на поле. Это осуществимо благодаря двум сценариям, при которых можно изменить ориентацию.

1 Вариант когда корабль ещё не находится на поле, тогда необходимо выполнить следующие действия:

  • 1. Выберите корабль, щелкнув по нему левой кнопкой мыши.
  • 2. Перетащите выбранный корабль на игровое поле.
  • 3. После размещения корабля на поле наведите курсор на него и щелкните правой кнопкой мыши, чтобы изменить его ориентацию по часовой стрелке.

Обратите внимание, что между кораблями должна быть как минимум одна клетка, и они не могут стоять друг на друге. Если корабль не помещается в выбранную позицию, система не позволит разместить его в этом месте.

2 Вариант когда корабль уже размещён на поле пользователя, тогда необходимо выполнить следующие действия:

  • 1. Наведите курсор на корабль, который уже размещён на поле.
  • 2. Щелкните правой кнопкой мыши, чтобы изменить ориентацию корабля по часовой стрелке.
  • 3. После размещения корабля на поле наведите курсор на него и щелкните правой кнопкой мыши, чтобы изменить его ориентацию по часовой стрелке.

Обратите внимание, что между кораблями должна быть как минимум одна клетка, и они не могут стоять друг на друге. Если корабль не помещается в выбранную позицию, система не позволит разместить его в этом месте.

Случайная расстановка

Для того чтобы расставить корабли автоматически случайным образом пользователю необходимо нажать на кнопку- «Случайная». (см. рисунок ниже).

После нажатия на кнопку «Случайная» на игрвом поле пользователя осуществляется случайная расстановка кораблей. (см. рисунок ниже).

Расстановка «Берега»

«Берега» – это алгоритм расстановки кораблей при котром все корабли кроме однопалубных располагаются вдоль внешних клеток поля(по краям поля), а однопалубные корабли размещаются в оставшиеся доступные клетки поля.

Для расстановки кораблей автоматически по алгоритму «Берега» пользователю необходимо нажать кнопку «Берега» . (см. рисунок ниже).

После нажатия на кнопку «Берега» на игрвом поле пользователя осуществляется расстановка кораблей по выбранному алгоритму. (см. рисунок ниже).

Сохранение расстановки в файл

Независимо от того, каким способом пользователь расставил корабли на своем поле — автоматически с использованием выбранного алгоритма или вручную — он может сохранить свою расстановку в файл, для этого пользователю необходимо нажать кнопку "Сохранить в файл".(см. рисунок ниже). По нажати кнопки ситсема окроет окно "Проводника" где пользователю нужно выбрать расположение файла для сохранения.

Загрузить расстановку из файла

Для того, чтобы воспользоваться расстановками, которые пользователь сохранил на свой компьютер нужно нажать на кнопку «Загрузить из файла». После чего откроется файловая система компьютера, которая позволит найти нужный файл. (см. рисунок ниже).

Сбросить расстановку

Для того, чтобы убрать все корабли с поля можно использовать кнопку «Сбросить». Данная кнопка позволяет очистить поле от кораблей. (см. рисунок ниже).

Ход игры

Начало игры

Для того, чтобы начать игру после успешной расстановки кораблей нужно нажать на кнопку «Играть». (см. рисунок ниже).

После данного нажатия все кнопки, связанные с расстановкой кораблей на поле, будут скрыты системой и можно начинать игру. Если пользователь хочет прервать игру, то он сможет нажать на кнопку «Выйти». (см. рисунок ниже).

Рассмотрим игровой процесс.

Пользователь всегда первым начинает игру. Для выстрела по полю противника пользователю необходимо нажать левой кнопкой мыши на выбранную для выстрела клетку поля. (см. рисунок ниже).

Если пользователь попадает в корабль противника, то в клетке появится красный радар, если не попадает, то зеленый и ход переходит компьютеру. Под полем компьютера появится надпись: «Ход противника. Думает…» (см. рисунок ниже).

Когда компьютер делает свой ход и попадает в корабль пользователя, то в клетке появляется анимация огня , если нет, то знак голубого радара и ход переходит к пользователю. (см. рисунок ниже).

Игра заканчивается, когда одна из сторон уничтожает все корабли противника.

Итоги игры

После завершения игры подводятся итоги в новом окне. Кроме того, пользователь сможет нажать «Сыграть еще раз», чтобы сыграть новую игру или же нажать «Выйти», чтобы выйти из неё. Если победа за пользователем, то надпись в окне будет следующей. (см. рисунок ниже).

Если в игре победил компьютерный противник, то надпись будет "Вы проиграли (: )". (см. рисунок ниже).
